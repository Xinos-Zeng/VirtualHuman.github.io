# VirtualHuman.github.io

虚拟人类前端项目

## 项目简介

## 开发指南

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 部署到GitHub Pages

```bash
npm run deploy
```

## 项目结构

```
VirtualHuman.github.io/
├── public/             # 静态资源
│   └── index.html      # HTML模板
├── src/                # 源代码
│   ├── App.css         # 应用样式
│   ├── App.js          # 主应用组件
│   ├── index.css       # 全局样式
│   └── index.js        # 入口文件
├── .babelrc            # Babel配置
├── package.json        # 项目依赖和脚本
└── webpack.config.js   # Webpack配置
```
